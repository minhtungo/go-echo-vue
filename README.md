# Go-Echo-Vue App

## Usage
Go-Echo-Vue app is a small app used to get and add data to IPFS - InterPlanetary File System

## Project setup
```
npm install
```

### Run front-end
```
npm run serve
```

### Run backend-end
```
go run .
```

### Compiles and minifies for production
```
npm run build
```

### Run ipfs daemon
```
ipfs daemon
```
